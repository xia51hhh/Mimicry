# 画布交互设计

> **状态**: Draft | **最后更新**: 2026-04-17

---

## 概述

Mimicry 使用 [Vue Flow](https://vueflow.dev/) 作为画布引擎，提供可视化工作流编辑体验。本文档定义画布的交互规范，参考 ComfyUI 和 Automa 的交互模式。

---

## 画布基础操作

### 视图控制

| 操作 | 触发方式 | 说明 |
|------|---------|------|
| 平移 | 鼠标中键拖拽 / 空格+左键拖拽 | 移动画布视口 |
| 缩放 | 鼠标滚轮 / Ctrl+滚轮 | 缩放画布（范围 25%~400%） |
| 适配全部 | `Ctrl+Shift+F` | 自动缩放至显示所有节点 |
| 适配选中 | `Ctrl+Shift+W` | 缩放至选中节点区域 |
| 重置视图 | `Ctrl+0` | 恢复 100% 缩放 |

### 选择操作

| 操作 | 触发方式 | 说明 |
|------|---------|------|
| 选中单个 | 左键点击节点 | 选中并显示属性面板 |
| 多选 | Shift+左键点击 | 追加到选中集合 |
| 框选 | 左键拖拽空白区域 | 框内节点全部选中 |
| 全选 | `Ctrl+A` | 选中所有节点 |
| 取消选择 | 点击空白区域 / `Esc` | 清除选择 |

### 节点操作

| 操作 | 触发方式 | 说明 |
|------|---------|------|
| 移动节点 | 拖拽节点 | 自由移动位置 |
| 批量移动 | 拖拽选中的多个节点 | 保持相对位置移动 |
| 复制 | `Ctrl+C` | 复制选中节点 |
| 粘贴 | `Ctrl+V` | 粘贴到鼠标位置 |
| 剪切 | `Ctrl+X` | 剪切选中节点 |
| 删除 | `Delete` / `Backspace` | 删除选中节点及关联连线 |
| 复制节点 | `Ctrl+D` | 原地复制（偏移 20px） |
| 撤销 | `Ctrl+Z` | 撤销上一步操作 |
| 重做 | `Ctrl+Shift+Z` | 重做已撤销操作 |

## Block 添加方式

### 1. 侧边栏拖拽

从左侧 Block 面板拖拽到画布：

```
┌────────────────┐     拖拽     ┌──────────────────────────┐
│  Block 面板     │  ─────────► │        画布               │
│                │             │                          │
│  🎯 触发与控制  │             │    ┌──────────┐          │
│    Manual      │             │    │ Navigate │          │
│    Condition   │             │    └──────────┘          │
│  🌐 浏览器     │             │           ↓ 放置位置      │
│    Navigate    │             │    ┌──────────┐          │
│    New Tab     │             │    │ Click ★  │ ← 新添加  │
│  👆 页面交互   │             │    └──────────┘          │
│    Click       │             │                          │
│    Type        │             └──────────────────────────┘
│  📊 数据       │
│  ⚡ 高级       │
└────────────────┘
```

### 2. 快速搜索添加

双击画布空白区域或按 `/`，弹出搜索框：

```
┌──────────────────────────────────────┐
│  🔍 添加 Block...                     │
│  ──────────────────────────────────  │
│  click                               │
│  ┌────────────────────────────────┐  │
│  │  👆 Click        页面交互       │  │
│  │  👆 Double Click  页面交互      │  │
│  │  📷 Click Screenshot 数据      │  │
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘
```

- 输入关键字模糊搜索 Block 名称和描述
- 方向键导航，回车确认
- 新 Block 添加到双击/光标位置

### 3. 连线末端自动添加

从节点输出端口拖出连线，在空白区域释放时自动弹出搜索框，选择后创建新 Block 并自动连线。

## 连线操作

### 创建连线

```
从输出端口 ●─────拖拽─────────► 到输入端口 ●
           (显示虚线预览)
```

- 鼠标悬停端口时高亮显示
- 拖拽时显示虚线预览
- 释放到有效端口时创建连线
- 释放到空白区域时弹出快速添加（见上方）

### 断开连线

- 选中连线后按 `Delete`
- 右键连线 → "删除连线"
- 从已连接的输入端口拖出 → 断开原连线并开始新连线

### 重新连线

拖拽连线的端点到其他端口即可重新连接。

## 分组操作

### 创建分组

1. 框选多个节点
2. 右键 → "创建分组" 或 `Ctrl+G`
3. 输入分组名称和颜色

### 分组行为

```
┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐
│  📁 数据提取阶段              │
│                              │
│  ┌──────┐  ┌──────────────┐ │
│  │Click │─►│ Get Text     │ │
│  └──────┘  └──────────────┘ │
│                              │
└─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘
```

- 拖拽分组标题栏移动整个分组及内部节点
- 分组仅为视觉组织，不影响执行逻辑
- 与 [Package](../design/package-system.md) 不同：分组不可封装复用

## 辅助功能

### 网格与对齐

- 可选网格背景（默认开启，20px 间距）
- 拖拽节点时自动吸附网格
- 对齐辅助线：移动节点时显示与相邻节点的对齐参考线

### 小地图

画布右下角显示缩略导航图（minimap），可拖拽定位视口。
