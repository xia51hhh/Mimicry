## IPC йҖҡдҝЎ

еүҚз«ҜдёҺ RustгҖҒRust дёҺ Python д№Ӣй—ҙйҮҮз”ЁдёҚеҗҢзҡ„йҖҡдҝЎж–№ејҸпјҡ

```
Vue 3 WebView  в”Җв”Җinvoke()в”Җв”Җв–ә  Rust Core  в”Җв”Җstdio JSON-RPCв”Җв”Җв–ә  Python Sidecar
               в—„в”Җв”Җevent()в”Җв”Җ              в—„в”Җв”Җв”Җresponseв”Җв”Җв”Җ
```

### еүҚз«Ҝ вҶ” Rust

Tauri v2 зҡ„ `invoke()` / `emit()` жңәеҲ¶пјҡ

```typescript
// еүҚз«Ҝи°ғз”Ё Rust е‘Ҫд»Ө
const result = await invoke('browser_navigate', { url: 'https://example.com' });

// еүҚз«Ҝзӣ‘еҗ¬ Rust дәӢд»¶
listen('browser-status', (event) => {
  console.log(event.payload);
});
```

### Rust вҶ” Python Sidecar

JSON-RPC 2.0 over stdioпјҡ

```json
// Request (Rust вҶ’ Python)
{"jsonrpc": "2.0", "id": 1, "method": "browser.navigate", "params": {"url": "https://example.com"}}

// Response (Python вҶ’ Rust)
{"jsonrpc": "2.0", "id": 1, "result": {"success": true, "url": "https://example.com"}}
```
