## 三层调试体系

```
┌─────────────────────────────────────────────────┐
│                 调试系统分层                       │
│                                                 │
│  Layer C: 实时观察 ──────────────── MVP ✅        │
│  ┌─────────────────────────────────────────┐    │
│  │ 节点执行状态高亮 + 数据摘要浮窗            │    │
│  └─────────────────────────────────────────┘    │
│                                                 │
│  Layer A: Block 级断点 ────────── MVP ✅         │
│  ┌─────────────────────────────────────────┐    │
│  │ 在指定 Block 设置断点，运行至此暂停        │    │
│  │ 暂停时可查看上下文数据，手动继续/终止       │    │
│  └─────────────────────────────────────────┘    │
│                                                 │
│  Layer B: 步进模式 ────────────── 后续版本       │
│  ┌─────────────────────────────────────────┐    │
│  │ 逐 Block 单步执行，每步暂停等待继续        │    │
│  └─────────────────────────────────────────┘    │
└─────────────────────────────────────────────────┘
```

## Layer C: 实时观察（MVP）

### 节点执行状态可视化

工作流运行时，画布上的每个 Block 节点实时显示执行状态：

| 状态 | 视觉表现 | 说明 |
|------|---------|------|
| `idle` | 默认样式（灰色边框） | 未执行 |
| `running` | 蓝色脉冲动画 + 蓝色边框 | 正在执行 |
| `success` | 绿色边框 + ✅ 图标 | 执行成功 |
| `error` | 红色边框 + ❌ 图标 | 执行失败 |
| `skipped` | 虚线边框 + 灰色 | 被跳过（disabled 或条件分支未命中） |
| `paused` | 黄色边框 + ⏸ 图标 | 断点暂停中 |

### 连线动画

执行时连线显示数据流动动画（流动虚线），当前活跃路径高亮：

```
 [Navigate] ===流动动画===> [Click] ───灰色──── [Get Text]
   ✅ 完成                  🔵 执行中             ⚪ 待执行
```

### 数据摘要浮窗

鼠标悬停在已执行的节点上时，显示数据摘要 Tooltip：

```
┌──────────────────────────────┐
│  Click #submit-btn           │
│  ─────────────────────────── │
│  状态: ✅ 成功                │
│  耗时: 234ms                 │
│  输出: { clicked: true }     │
│  选择器: #submit-btn         │
│  匹配元素: 1                 │
└──────────────────────────────┘
```

## Layer A: Block 级断点（MVP）

### 设置断点

- **画布操作**：右键 Block → "切换断点" / 点击节点左侧断点区域
- **快捷键**：选中节点后按 `F9`
- **视觉标记**：节点右上角显示红色圆点 🔴

### 断点行为

```
正常执行 → 到达断点节点 → 暂停
                          │
                          ▼
                  ┌──────────────┐
                  │  暂停状态     │
                  │              │
                  │  可查看:      │
                  │  - 变量值     │
                  │  - 前节点输出  │
                  │  - 浏览器状态  │
                  │  - 循环进度    │
                  │              │
                  │  操作:        │
                  │  [▶ 继续]    │
                  │  [⏭ 跳过]    │
                  │  [⏹ 终止]    │
                  └──────────────┘
```

### 断点暂停时的数据面板

暂停时自动打开 Data Inspector 面板（见下方），展示当前执行上下文的完整数据快照。

---

## Layer B: 步进模式（后续版本）

> 此功能不在 MVP 范围内，设计预留。

- 点击工具栏 ⏭ Step 按钮或 `F10` 快捷键
- 每执行一个 Block 后自动暂停
- 支持 Step Into（进入 Package 内部）和 Step Over（跳过 Package）

独立的调试数据查看面板，显示工作流运行时的详细数据：

```
┌───────────────────────────────────────────────────┐
│  Data Inspector                            [📌]   │
│                                                   │
│  ┌─ Variables ──────────────────────────────────┐ │
│  │  baseUrl: "https://example.com"              │ │
│  │  retry: 0                                    │ │
│  │  results: Array(15)                          │ │
│  └──────────────────────────────────────────────┘ │
│                                                   │
│  ┌─ Current Node Output ────────────────────────┐ │

│  │  {                                           │ │
│  │    "text": "iPhone 15 Pro",                  │ │
│  │    "selector": ".product-title"              │ │
│  │  }                                           │ │
│  └──────────────────────────────────────────────┘ │
│                                                   │
│  ┌─ Loop Context ───────────────────────────────┐ │
│  │  loopId: loop_products                       │ │

│  │  current: { title: "iPhone 15 Pro", ... }    │ │
│  └──────────────────────────────────────────────┘ │
```

### 功能

- **树形展开**：复杂对象/数组可逐层展开查看
- **搜索过滤**：输入关键字过滤变量名
- **复制值**：点击值右侧复制按钮，复制到剪贴板
- **固定面板**：点击 📌 固定在侧边栏

## 执行日志

底部面板显示执行日志流，实时追加：

```
┌───────────────────────────────────────────────────────────────┐
│  执行日志                                    [🔍] [🗑 清除]   │
│                                                               │
│  12:00:01.234  ✅  node_1  Manual Trigger      started        │
│  12:00:01.235  ✅  node_2  Navigate             https://...   │
│  12:00:02.891  ✅  node_3  Loop Elements         25 elements  │
│  12:00:03.012  ✅  node_4  Get Text             "iPhone 15.." │
│  12:00:03.156  ⚠️  node_5  Get Text             selector 自愈  │
│  12:00:03.200  ✅  node_6  Loop Breakpoint       iter 1/25    │
│  12:00:03.450  ❌  node_4  Get Text             timeout 5000  │
│  ...                                                          │
└───────────────────────────────────────────────────────────────┘
```

### 日志级别

| 级别 | 图标 | 场景 |
|------|------|------|
| `info` | ✅ | 正常执行完成 |
| `warn` | ⚠️ | 选择器自愈、重试成功 |
| `error` | ❌ | 执行失败 |
| `debug` | 🔍 | 详细调试信息（需开启 debugMode） |
