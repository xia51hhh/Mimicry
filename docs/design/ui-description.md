## 一、设计原则

1. **内容优先** — 参考 Notion 的留白和清晰排版，UI 元素不喧宾夺主
2. **速度感** — 参考 Linear 的极速交互反馈，操作 → 响应延迟感知为零
3. **可发现性** — 功能不藏在深层菜单中，常用操作 ≤ 2 次点击可达
4. **一致性** — 同类交互行为保持一致的视觉反馈和动效模式
5. **可定制** — 面板位置、大小、显隐由用户自由控制，主题色可切换

## 二、整体布局架构

Mimicry 采用 **VS Code 式四区布局**，所有面板均可自由拖拽调整大小、折叠隐藏。

```
┌──────────────────────────────────────────────────────────────────┐
│  顶栏: [工作空间选择] [Tab1 Tab2 Tab3 +] [⚙设置] [用户头像]     │
├──────────┬───────────────────────────────────────┬───────────────┤
│          │                                       │               │
│  左侧栏  │           中央主区域                    │  右侧属性面板  │
│  (Block  │   ┌─────────────────────────────┐     │  (Block 配置  │
│   面板)  │   │     Vue Flow 工作流画布      │     │   / 数据)    │
│          │   │                             │     │               │
│  可折叠   │   │                             │     │  可折叠        │
│  可拖拽   │   └─────────────────────────────┘     │  可拖拽        │
│          │   ┌─────────────────────────────┐     │               │
│          │   │ 底部面板 (Monaco/日志/变量)   │     │               │
│          │   └─────────────────────────────┘     │               │
├──────────┴───────────────────────────────────────┴───────────────┤
│  状态栏: [浏览器状态] [Block数] [执行状态] [Sidecar] [版本]       │
└──────────────────────────────────────────────────────────────────┘
```

### 尺寸规格

| 区域 | 默认尺寸 | 可调范围 | 行为 |
|------|---------|---------|------|
| 顶栏 | 高 44px | 固定 | 始终可见 |
| 左侧栏 | 宽 260px | 180-400px | Ctrl+B 折叠/展开，双击分隔线恢复默认 |
| 右侧属性面板 | 宽 320px | 240-500px | 无选中节点时自动折叠，双击分隔线恢复默认 |
| 底部面板 | 高 200px | 100-50%窗口 | Ctrl+J 折叠/展开 |
| 状态栏 | 高 24px | 固定 | 始终可见 |
| 中央画布 | 自适应 | 填充剩余空间 | — |

### 面板拖拽规则

- 拖拽面板分隔线（3px 宽的透明热区）调整大小
- 拖拽时显示蓝色高亮线指示当前宽度
- 松开后平滑过渡到新尺寸（spring 动效 120ms）
- 双击分隔线 → 恢复该面板默认尺寸（spring 动效）
- 拖拽至极小值（< 最小宽度）→ 面板自动折叠

## 三、顶栏 — Tab 工作区 + 导航

顶栏是全局导航的核心，高度 44px，设计参考浏览器 Tab 栏。

### 布局（从左到右）

```
┌─────────────────────────────────────────────────────────────────┐
│ [📁▼ My Workspace] │ [📄Tab1 ✕] [📄Tab2 ✕] [📄Tab3 ✕] [+] │ [▶ 运行] [🔴 录制] │ [⚙] [👤] │
└─────────────────────────────────────────────────────────────────┘
  ↑ 工作空间选择器       ↑ Tab 栏                                    ↑ 操作区       ↑ 系统区
```

### 工作空间选择器（左端）

- 下拉按钮，显示当前工作空间名称
- 点击展开下拉菜单：
  - 最近打开的工作空间列表（最多 5 个）
  - 分隔线
  - "新建工作空间"
  - "打开工作空间..."（系统文件对话框）
- 工作空间 = 一个目录，包含多个 .mimicry 工作流文件

### Tab 栏（中部，占据主要空间）

- 每个 Tab 代表一个打开的工作流文件
- Tab 样式：文件图标 + 名称 + 关闭按钮 ✕
- 活跃 Tab：底部 2px 蓝色指示条 + 文字高亮
- 非活跃 Tab：灰色文字，hover 显示关闭按钮
- Tab 可拖拽重新排序
- Tab 右键菜单：关闭 / 关闭其他 / 关闭右侧所有 / 复制路径
- 最右侧 [+] 按钮：新建工作流
- Tab 过多时：左右滚动箭头出现，或鼠标滚轮横向滚动
- 未保存的 Tab：名称后显示圆点 ● 标记
- 双击 Tab 名称：重命名工作流

### 操作区

- **▶ 运行**（绿色按钮）：执行当前工作流
  - 运行中变为 ⏹ 停止（红色）
  - 按钮旁有下拉箭头：运行 / 调试运行 / 从选中节点运行
- **🔴 录制**：开始浏览器录制
  - 录制中变为红色闪烁 + 计时器显示
- **🌐 浏览器**：浏览器状态指示灯
  - 绿色 = 已连接，灰色 = 未连接
  - 点击：已连接时前置浏览器窗口，未连接时启动浏览器

### 系统区（右端）

- **⚙ 设置齿轮**：点击进入设置页面（新 Tab 打开）
- **👤 用户头像/图标**：预留（后续版本可接入账号系统）

## 四、左侧栏 — Block 面板

### 搜索框

- 顶部固定搜索框，placeholder: "搜索 Block..."
- 实时模糊匹配 Block 名称和描述
- 搜索结果扁平显示（不分类），高亮匹配文字
- 清除按钮 / Esc 恢复分类列表

### Block 分类列表

手风琴面板，每类左侧有 3px 彩色指示条。完整列表与 [block-system.md](./block-system.md) 严格对齐：

#### 🎯 触发与控制（蓝色 #3B82F6）

Manual Trigger、Schedule Trigger、Condition、Loop Data、Loop Elements、Repeat、While Loop、Loop Breakpoint、Delay、Wait Connections、Stop

#### 🌐 浏览器操作（绿色 #10B981）

Navigate、New Tab、Switch Tab、Go Back、Go Forward、Reload、Close Tab、Switch Frame、Wait For Page、Handle Dialog

#### 👆 页面交互（橙色 #F59E0B）

Click、Type、Hover、Press Key、Scroll、Select Option、Upload File、Clear、Focus

#### 📊 数据处理（紫色 #8B5CF6）

Get Text、Get Attribute、Get URL、Screenshot、Extract Table、Set Variable、Transform、Export、Cookie

#### ⚡ 高级（红色 #EF4444）

Run Script、HTTP Request、Execute Workflow、Package、Log、Comment

### 单个 Block 条目

```
┌─────────────────────────┐
│ 🖱 Click          👆    │  ← 图标 + 名称 + 分类色标
│ 点击页面元素             │  ← 描述（hover时淡入）
└─────────────────────────┘
```

- 默认只显示图标 + 名称（单行）
- 鼠标悬停 300ms 后展开描述行
- 点击 → 添加到画布当前视口中心
- 拖拽 → 拖到画布指定位置释放

### 底部

- **📦 Package 库**（折叠面板）：用户创建的 Package 列表，可拖入画布
- **⏱ 最近使用**（折叠面板）：最近 5 个使用的 Block 快捷入口

## 五、中央画布 — 工作流编辑器

使用 Vue Flow 渲染，占据最大可用空间。

### 画布背景

- 深色主题：细密点阵网格（#2A2A3C 点，间距 20px）
- 浅色主题：浅灰点阵网格（#E5E5E5 点）
- 无限平移 + 缩放（25%-400%）

### 基础操作

| 操作 | 触发 |
|------|------|
| 平移 | 中键拖拽 / 空格+左键拖拽 |
| 缩放 | 滚轮 / Ctrl+滚轮 |
| 适配全部 | Ctrl+Shift+F |
| 适配选中 | Ctrl+Shift+W |
| 重置缩放 | Ctrl+0 |
| 框选 | 左键拖拽空白区域 |
| 多选 | Shift+点击 |
| 全选 | Ctrl+A |
| 删除 | Delete / Backspace |
| 复制/剪切/粘贴 | Ctrl+C / X / V |
| 撤销/重做 | Ctrl+Z / Ctrl+Shift+Z |

### Block 节点渲染

每个 Block 渲染为圆角矩形卡片（圆角 8px，阴影 subtle）：

```
       ● input
┌─────────────────────┐
│ 🌐 Navigate          │  ← 分类色条(左侧3px) + 图标 + 名称
│─────────────────────│
│ URL: {{$var.url}}   │  ← 关键参数摘要（最多2行，溢出省略）
│ Wait: networkidle   │
└─────────────────────┘
       ● output
```

**视觉状态**：

| 状态 | 表现 |
|------|------|
| 默认 | 暗色卡片，白色文字 |
| 悬停 | 卡片微微上浮（translateY -1px），边框亮度提升 |
| 选中 | 蓝色边框 2px，轻微发光 |
| 运行中 | 蓝色边框呼吸动画（opacity 0.5→1 循环 1.5s） |
| 成功 | 左侧色条变绿，3s 后淡回默认 |
| 失败 | 左侧色条变红 + 红色感叹号徽标 |
| 断点 | 右上角红色圆点 🔴 |
| 禁用 | 半透明 50% + 虚线边框 |
| Package | 双层边框 + 右下角 📦 展开图标 |

### 连接线

- 默认：灰色贝塞尔曲线（#4A4A5A，1.5px）
- 悬停：蓝色高亮
- 数据流向：微弱流动点动画（小圆点 3px 沿线移动，2s 周期）
- 条件分支：true 路径绿色标签，false 路径红色标签
- 执行中：活跃线变绿 + 加速流动
- 失败路径：线变红色

### 右键菜单

**空白区域右键**：添加 Block（弹出搜索面板）| 粘贴 | 全选 | 适应视图

**节点右键**：复制 | 剪切 | 删除 | 设置断点 | 启用/禁用 | 从此运行 | 创建 Package（多选）| 创建分组（多选）

### 快速添加

双击画布空白处，或按 `/` 键：
- 在点击位置弹出浮动搜索框
- 实时搜索 Block 名称
- 回车或点击选择 → 在该位置创建新 Block
- Esc 关闭

从输出端口拖出连线释放到空白处 → 自动弹出搜索框，选择后创建并自动连线。

### 画布覆盖层

**右下角（浮动工具栏）**：
- 缩放控制：[−] [100%] [+] [适应全部]
- 小地图开关（MiniMap）

**小地图**：
- 默认隐藏，点击开关后显示在右下角
- 半透明矩形缩略图，显示所有节点和当前视口范围
- 可拖拽视口矩形快速导航

### 分组

- 框选多节点 → 右键 → "创建分组"（Ctrl+G）
- 分组显示为虚线矩形 + 标题栏 + 可选颜色
- 拖拽分组标题移动整组节点
- 分组仅为视觉组织，不影响执行（与 Package 区别：Package 可封装复用为独立节点）

## 六、右侧属性面板

### 面板行为

- 无选中节点时：面板显示工作流全局设置（名称、描述、全局变量概览）
- 选中节点后：spring 动效切换为该 Block 的配置面板
- 选中多个节点：显示批量操作面板（批量启用/禁用、批量设置错误处理）

### 面板头部

```
┌─────────────────────────────────┐
│ 👆 Click                  [✓ 启用]│
│ node_3          interaction/Click │
└─────────────────────────────────┘
```

### Tab 切换

面板顶部两个 Tab（下划线指示器，切换有 slide 动效）：

- **设置** — Block 配置参数（默认）
- **数据** — 上次执行该节点的输入/输出数据（JSON 树形视图）

### 配置区域（设置 Tab）

按 Block 类型动态渲染表单。示例 — Click Block：

**📌 元素选择器区域**
- 选择器输入框 + 类型下拉（CSS/XPath/text=/role=）
- 输入框右侧 🎯 按钮：进入选择器模式
- 输入框下方实时验证：匹配数量 + 唯一性状态（✅ 唯一 / ⚠️ 多个 / ❌ 无匹配）
- 备选选择器折叠区（展开显示 fallback 列表）

**⚙ 操作选项**（Block 特定）
- Click 类型（单击/双击/右键）
- 等待元素出现 + 超时时间

**🔄 错误处理**（所有 Block 通用）
- 策略下拉：Stop / Continue / Retry / Fallback / Inherit
- 重试次数 + 重试间隔（仅 Retry 时可见）

**📝 备注**
- 可折叠文本区域

### 各 Block 配置概要

| Block | 关键配置字段 |
|-------|------------|
| Navigate | URL 输入、等待策略（load/domcontentloaded/networkidle） |
| Type | 选择器、输入文本（支持 `{{$var}}`）、清空选项、输入延迟 |
| Loop Data | 数据源（变量/JSON数组/表格）、Loop ID、最大迭代数 |
| Condition | 条件构建器（操作数+运算符+操作数）、AND/OR 组合 |
| HTTP Request | 方法、URL、Headers 编辑器、Body 编辑器、响应变量 |
| Handle Dialog | 弹窗类型（alert/confirm/prompt）、操作（接受/拒绝/输入） |
| Run Script | 语言选择（JS/Python）、内嵌代码编辑器（Monaco mini） |

## 七、底部面板 — Monaco / 日志 / 变量

底部面板通过 Tab 切换三个子视图。

### Tab 1: JSON 编辑器（Monaco Editor）

- 显示当前工作流完整 JSON
- 双向实时同步：JSON 编辑 ↔ 画布
- JSON Schema 校验：错误处红色波浪线
- 语法高亮 + 代码折叠 + 搜索替换（Ctrl+F）
- LLM 场景：粘贴 LLM 生成的 JSON → 自动解析 → 画布更新

### Tab 2: 执行日志

- 实时滚动日志流
- 每行：`[时间] [Block名] [事件] [数据摘要]`
- 颜色编码：🟢成功 / 🔴错误 / 🟡警告 / ⚪信息
- 点击日志行 → 画布平移定位到对应 Block + 闪烁高亮
- 过滤器栏：按 Block 名称 / 按级别筛选
- 清空按钮

### Tab 3: 变量监控

- 表格：变量名 | 类型 | 当前值 | 最后修改 Block
- 运行时值实时更新（更新时该行短暂绿色背景闪烁 300ms）
- 双击值单元格 → 编辑（调试用）
- [+ 新增变量] 按钮
- 支持搜索过滤

## 十、元素选择器交互

### 触发

1. 点击 Block 属性面板选择器输入框旁 🎯 按钮
2. 工具栏选择器按钮
3. 快捷键 Ctrl+Shift+E

### 浏览器叠加层

进入选择模式后，Camoufox 窗口前置，注入高亮叠加层：

- 鼠标移动时：当前悬停元素蓝色半透明高亮框（border 2px + 背景 rgba 10%）
- 元素上方悬浮标签：`<button.submit-btn>` （tag + 关键class/id）
- 窗口底部 DOM 路径面包屑

### 点击选择后 — 智能分析浮窗

```
┌──────────────────────────────────────────────────┐
│  选择器推荐                                        │
│  ──────────────────────────────────────────────── │
│  ⭐⭐⭐ #submit-btn              CSS    ✅ 唯一    │ ← 选中态（蓝色背景）
```

## 十九、微动效系统

参考 Material Design 3 的 spring 物理系统 + NNGroup 动效设计原则。所有动效追求"快速、有意义、不干扰"。

### 动效分层

| 类型 | 用途 | 时长 | 缓动 |
|------|------|------|------|
| **反馈动效** | 按钮点击、开关切换、hover 状态 | 80-120ms | spring(stiffness:400) |
| **切换动效** | Tab 切换、面板折叠、内容替换 | 150-200ms | ease-out / spring |
| **导航动效** | Package 钻入/钻出、设置页进入 | 250-350ms | spring(stiffness:200, damping:20) |
| **状态动效** | 执行脉冲、成功/失败指示 | 持续性 | linear / ease-in-out |

### 具体动效清单

#### 按钮与控件
- hover：背景色渐变 80ms
- 点击：scale 0.97 → 1.0（spring 120ms）
- 开关切换：圆形滑块滑动 + 背景色过渡（150ms spring，微弱 overshoot）
- 下拉菜单展开：scaleY 0→1 + opacity 0→1（origin top, 150ms ease-out）
- 下拉菜单收起：opacity 1→0（80ms）

#### 面板
- 分隔线拖拽：实时响应，无延迟（requestAnimationFrame）
- 面板折叠：width/height 收缩到 0 + opacity 淡出（200ms ease-out）
- 面板展开：width/height 从 0 展开 + opacity 淡入（200ms ease-out）

#### 画布
- 节点拖拽：直接跟随鼠标，无动效延迟
- 节点放置（从侧栏拖入）：scale 0.8→1.0 + opacity 0→1（spring 200ms）
- 节点删除：scale 1.0→0.8 + opacity 1→0（150ms ease-in）
- 连线创建：端点跟随鼠标，线条实时贝塞尔更新
- 连线完成：线条从虚线变实线（100ms）
- 缩放：平滑插值（使用 Vue Flow 内置 transition）

#### 数据与反馈
- 执行进度脉冲：border-color 蓝色明暗交替（1.5s ease-in-out infinite）
- 成功指示：色条从默认→绿色（200ms）→ 3s 后→默认（500ms ease-out）
- 失败指示：色条从默认→红色（200ms）+ 轻微 shake 动效（translateX ±2px, 300ms）
- 变量更新闪烁：行背景色 transparent→green(10%)→transparent（300ms）
- Toast 通知：从底部 slide-up + fade-in（200ms），5s 后 fade-out（300ms）

### prefers-reduced-motion

当系统开启减少动效或用户在设置中关闭动画：
- 所有 transition/animation duration 降为 0ms
- 脉冲/呼吸动画替换为静态高亮
- 仅保留关键状态变化（颜色切换保留，位移和缩放移除）

## 二十二、术语约定

| 术语 | 含义 |
|------|------|
| Block | 工作流画布上的最小可执行单元（UI 面向用户时统一使用） |
| 节点 | Block 的同义词（技术文档中可互换） |
| Package | 多个 Block 封装为单个可复用节点 |
| 工作空间 | 包含多个工作流文件的目录 |
| 工作流 | 一个 .mimicry 文件，包含 Block 图 + 变量 + 配置 |
| Tab | 顶栏中打开的工作流 / 设置页面 |
